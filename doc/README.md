# Documentation du Projet Site NDI 2025

Bienvenue dans la documentation du projet **Site NDI 2025**. Ce projet est une application web interactive comprenant plusieurs mini-jeux et une interface moderne.

## Table des Matières

1. [Installation et Démarrage](./SETUP.md) - Comment installer et lancer le projet (Frontend et Backend).
2. [Architecture](./ARCHITECTURE.md) - Vue d'ensemble de la stack technique et de l'architecture.
3. [Les Jeux](./GAMES.md) - Détails sur les mini-jeux implémentés (Snake, Volume, RPG, LLM).
4. [Backend & API](./BACKEND.md) - Documentation du serveur Express et de l'intégration OpenRouter.
5. [Composants](./COMPONENTS.md) - Liste et description des principaux composants Vue.
6. [Logique & Stores](./LOGIC.md) - Détails sur la gestion d'état (Pinia) et les Composables.

## Structure du Projet

- `src/` : Code source du Frontend (Vue.js).
- `server/` : Code source du Backend (Node.js/Express).
- `doc/` : Cette documentation.
